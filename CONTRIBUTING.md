# Contributing

Thank you for your interest in this project.

⚠️ **Important**: This project is **NOT open source** and is licensed for
**personal use only**.

## What is allowed

- You may:
  - Use the project for personal purposes
  - Modify the code locally for your own personal use
  - Report bugs or issues

## What is NOT allowed

- You may NOT:
  - Submit pull requests containing code
  - Redistribute this project or modified versions
  - Use this project for commercial purposes
  - Offer this project as a service (SaaS, paid tools, subscriptions)
  - Re-license this project

## Bug reports

If you find a bug or issue, you may:
- Open an issue with a clear description
- Include logs or screenshots if helpful

Please do **NOT** submit pull requests with code changes.
